# Samsung Project — Приложение для анализа недвижимости

Десктопное приложение для определения типа недвижимости на основе параметров квартиры.

Программа анализирует введённые характеристики объекта и определяет, к какому сегменту рынка он относится.

---

## О приложении

Приложение предназначено для быстрого анализа квартир на основе обученной модели машинного обучения.

Пользователь вводит параметры недвижимости через графический интерфейс, после чего приложение:

* определяет тип объекта;
* относит квартиру к одному из рыночных сегментов;
* показывает результат анализа;
* может выявлять нетипичные объекты.

Приложение ориентировано на демонстрацию практического применения машинного обучения для анализа недвижимости.

---

## Возможности

* Ввод параметров квартиры через GUI
* Анализ объекта недвижимости
* Определение сегмента рынка
* Проверка на аномальность
* Использование обученной ML-модели

---

## Интерфейс приложения

Приложение представляет собой десктопный интерфейс с формой ввода параметров квартиры.

Пользователь может указать:

* стоимость;
* площадь;
* количество комнат;
* этаж;
* этажность дома;
* год постройки;
* время до метро;
* тип жилья.

После заполнения формы приложение выполняет анализ и выводит результат.

## Как пользоваться

### 1. Запуск приложения

```bash
python main.py
```

### 2. Ввод параметров

Введите характеристики квартиры в поля интерфейса.

### 3. Анализ

Нажмите кнопку анализа объекта.

### 4. Результат

Приложение отобразит:

* сегмент недвижимости;
* результат классификации;
* информацию об аномальности объекта.
